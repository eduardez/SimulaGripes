# SimulaGripazos
Práctica para la asignatura de diseño de algoritmos.

Para ejecutar:
```sh
$ java -jar <no_me_acuerdo_del_nombre_que_le_he_puesto.jar>
```

> El proyecto usa la libreria MigLayout, osea que es importante tenerla.
